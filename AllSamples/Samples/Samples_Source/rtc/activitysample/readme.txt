========================================================================
        ActivitySample Sample (Using RTC Client API)  
========================================================================

   
Sample Name:
=============

	ActivitySample.exe  [ActivitySample]


Sample Description:
=================

	ActivitySample Sample is an application written in C# that uses
        the RTC Core API to establish an Audio/Vedio call with any peer.
	
Configurations & Setup:
=======================
      
        - OS       = "Longhorn"
  

How To Run The Sample:
======================

- To execute the ActivitySample.exe, type ActivitySample.exe in the same directory 
  as the executable.

- To make a new call, enter the peer address in the field provided and click on 
  "Call" button.

- To terminate an existing call, click on "Stop" button.

- The status of the call will be displayed in the Status area of the main window.

- Disable ICF to make this sample run.
