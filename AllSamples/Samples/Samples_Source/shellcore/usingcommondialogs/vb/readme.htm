Using Common Dialogs (VB .NET) Sample

Summary

Performs file open and file save operations and also adds a control to the dialog.

The major techniques covered by this sample are:

  • Using the OpenFileDialog, SaveFileDialog, and FileDialogPushButton classes.

This document covers the following topics.

  • How to Build the Sample
  • How to Use the Sample
  • Code Tour

How to Build the Sample

Open the ComdlgSample.csproj file in Microsoft Visual Studio. Compilation of the comdlgsample.cs file produces a console application.

How to Use the Sample

The sample displays a control which has a text box and three pushbuttons: Open, Save, and Clear Text.

Clicking Open displays an Open dialog. The dialog displays either text files or all files.

Clicking Save displays a Save As dialog. Any characters in the text box are the entry for "File name:". If the text box is empty, "mydoc" is the entry. The dialog displays either text files or all files. Also, clicking the pushbutton labeled "Number of characters" displays the number of characters in the text box.

Clicking Clear Text clears the text box.
