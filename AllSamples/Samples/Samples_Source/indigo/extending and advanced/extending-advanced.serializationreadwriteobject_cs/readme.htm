Overview

The "Longhorn" communications subsystem is code-named "Indigo". The SerializationReadWriteObject sample is a basic serialization application that writes an object to a stream (serializes) and then reads the object back in from the stream (deserializes). This sample demonstrates the use of the WriteObject and ReadObject methods to control the root element serialization.

Key Concepts

Building

  1. Open the Longhorn Build Environment console window (click Start, click All Programs, click Microsoft Longhorn SDK, click Open Build Environment Window, and click Longhorn Build Environment).
  2. Navigate to the directory where this readme is located.
  3. Type MSBuild and press ENTER.

Running

To run the sample, open a console prompt in the CS directory contained within the SerializationReadWriteObject sample directory and type ReadWriteObject.
