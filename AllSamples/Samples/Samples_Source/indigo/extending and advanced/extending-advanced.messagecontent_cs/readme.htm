Overview

The "Longhorn" communications subsystem is code-named "Indigo". The MessageContent sample demonstrates how to create and use a custom MessageContent.

Key Concepts

To create custom content, one must inherit from the abstract MessageContent class. In this sample, ElementListContent derives from MessageContent.  ElementListContent exposes XmlElement[] Elements, which allows the user to access the multiple body parts as individual XmlElements.  In this sample, an XmlDocument is created from a string, and the XmlElements from the resulting DOM are put into an ElementListContent.  A message with this content is then sent from the Port.

Building

1.      Open the Longhorn Build Environment console window (click Start, click All Programs, click Microsoft Longhorn SDK, click Open Build Environment Window, and click Longhorn Build Environment).

2.      Navigate to the directory where this readme is located.

3.      Type MSBuild and press ENTER.

Running

To see the output, run the executable (CustomContentSample.exe) created in the source directory from the command prompt.
